Blockchain, as name it self says, chain. Block chain is nothing but series of blocks that link to one another and each block has unique value with which it is identified (it's called HASH value).

Let's break the process into small modules...

  • Block : the container of your data (eg: bitcoin data)
  • Previous Hash : it is pointer that points to hash value of previous block
  • Hash Value : The Unique value of the block, used to identify block or verify integrity
  • Time Stamp : timestamp is very important, as it is also helps detect modifying hash value.
  • GENESIS BLOCK : this is first most block or simply origin.
    •     Note: Previous Hash value for Genesis block is null/zero
